# House Builder App

A browser-based 3D room and house builder prototype built with Vite and Three.js.

## Features

- Isometric/orbit, top, walk, shell, roof, and tour viewing modes
- Room modules with adjustable size, wall color, floor color, and exterior finishes
- Drag-and-place furniture, decor, pets, kitchen, bathroom, outdoor, deck, door, window, stair, and roof assets
- Object selection, rotation, scaling, locking, variants, copy/paste, duplicate, and multi-select movement
- Local named project save/load through browser storage

## Getting Started

```bash
npm install
npm run dev
```

Open the local URL shown by Vite, usually:

```text
http://127.0.0.1:5173/
```

## Build

```bash
npm run build
```

The production build outputs to `dist/`, which is intentionally ignored by Git.
